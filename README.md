# silver-bullet

目录结构：

本系统工程采用maven进行代码管理
silver-bullet
amain       //存放WEB部分内容
cms-core     //通用的文章管理core，包括对文件的管理、文章类别管理，包括评论、回复
code-generator  //基于mybatic的代码自动生成
doc           //文档
sys-core     //存放配置、通用工具等
sys-auth     //存放用户、机构、菜单管理等
sys-log      //日志管理
sys-message  //消息模块，集成activemq
sys-params   //数据字典模块
sys-tag      //标签模块
ztest        //测试demo模块
